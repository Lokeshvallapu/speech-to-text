# speech-to-text
Speech to Text Transcription using Flask, Whisper AI, and SpeechRecognition. Supports audio file, and video transcription.

## Features
- Audio file transcription
- Video transcription
- Whisper AI integration
- Flask web interface

## Technologies Used

- Python
- Flask
- OpenAI Whisper
- SpeechRecognition
- MoviePy
- HTML / CSS / JavaScript

## How to Run

1. Clone the repository
 git clone "https://github.com/username/speech-to-text.git"

2. Navigate to the project folder 
    cd speech-to-text

3. Install dependencies
  pip install -r requirements.txt

4. Run the application
   python app.py

5. Open in browser
   "http://127.0.0.1:0000"
